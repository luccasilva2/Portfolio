# Portfolio de Lucca Oliveira

## Descrição
Este é o portfólio pessoal de Lucca Oliveira, desenvolvedor web full stack. O site apresenta suas habilidades, projetos e informações de contato de forma moderna e responsiva.

## Funcionalidades
- **Seção Hero**: Vídeo de fundo com apresentação pessoal
- **Sobre Mim**: Informações sobre experiência e paixão por tecnologia
- **Habilidades**: Tecnologias dominadas com níveis de proficiência
- **Projetos**: Showcase de projetos desenvolvidos
- **Contato**: Formulário de contato e links para redes sociais

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica
- **CSS3**: Estilização avançada e responsividade
- **JavaScript**: Interatividade
- **Remixicon**: Ícones
- **Google Fonts**: Tipografia (Inter)

## Como Visualizar
1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. O site é totalmente responsivo e funciona em dispositivos móveis

## Estrutura do Projeto
```
portfolio/
```
